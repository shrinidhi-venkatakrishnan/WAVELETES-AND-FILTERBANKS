To run the code:

1. Change the present working directory (pwd) of MATLAB to folder 'Code_Gopi' , where the code is avaialble.
2. The 'main.m' contains everything for decomposition, reconstruction and denoising.
3. Run main.m to see the results.
4. All the functions developed for the task are well commented.
	type 'help <functionName>' to get to know how to use the functions.
